## 📗프로젝트 소개

한 분야의 전문가가 되기 위해서는 1만 시간이 필요하다고 하는 법칙이 있으며 <br>
1만 시간을 채우기 위하여 하루에 몇시간씩 노력을해야 하는지 계산하기 위해서 만든 페이지입니다.
